# Hệ thống quản lý nhà hàng

## Giới thiệu
Dự án xây dựng hệ thống quản lý quán cà phê / nhà hàng sử dụng UML.

## Nội dung
- Phân tích hệ thống
- Thiết kế UML (Use Case, Class, Sequence, Activity, State)
- Mô phỏng quy trình hoạt động

## Tài liệu
Xem chi tiết trong file báo cáo đính kèm.
